# vue-ts

## 注意事项

### 切记只使用 npm 包管理器安装依赖(其他包管理器的 Prettier 依赖异常，导致代码格式化差异大)

### vscode 安装 Prettier formatter 插件，保存的同时可自动格式化，风格统一

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
